# Bug Evidence - BisouTrip

This folder contains screenshots and technical reports for bugs identified during the testing of the BisouTrip application.

## 📋 Reporting Format
When documenting a bug, please include:
- **Title**: Brief summary of the issue.
- **Steps to Reproduce**: Detailed steps.
- **Expected Result**: What should have happened.
- **Actual Result**: What actually happened.
- **Evidence**: Screenshots or logs.
